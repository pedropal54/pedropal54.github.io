Hosval

Descripción


Hosval es una aplicación (en local) que gestiona los pacientes afectados por el COVID-19 en el hospital de Valdepeñas. Su uso sería recomendable cuando hay saturación en los contagios y se lleva una mala gestión de los pacientes.

Desarrollo


Se ha utilizado el entorno de desarrollo Android Studio en el cual se programa con el lenguaje Java (muy importantes los objetos). En cuanto al desarrollo no fue difícil debido a que este entorno ofrece muchas ayudas en cuanto al diseño gráfico. Se tiene una base de datos (en local) para tener todos los pacientes con sus respectivos datos.

La aplicación cuenta con todo tipo de comprobación de errores. Se dispone de usuario y contraseña para su logeo. Cabe la posibilidad de buscar algún paciente específico (por cualquier cadena de caracteres de su nombre y apellidos) o de añadir un paciente nuevo. También se pueden modificar o eliminar los pacientes, que van ordenados por el color de su icono que representa su estado (rojo = riesgo, amarillo = con síntomas, verde = sin peligro).

Experiencia de aprendizaje


La verdad es que me llevo muy buenas vibraciones con este entorno de desarrollo y esta práctica. Con los profesores de mi asignatura (Interacción Persona-Ordenador II) también. Creo que es un proyecto muy interesante y que si se domina este entorno y lenguaje se puede llegar a diseñar cosas muy grandes.

Tu-rural

Descripción


Aplicación móvil que sirve para gestionar la rutas turísticas de Ciudad Real (Castilla la Mancha). Ha sido desarrollada con Visual Studio Code, utilizando el lenguaje Dart y el framework Flutter. La aplicación sólo funciona en local (salvo el usuario y contraseña que están en la nube gracias a Firebase).

Desarrollo


Fue un proyecto muy denso y largo, ya que partíamos de cero con Dart. Al ser un lenguaje orientado a objetos fue una gran ventaja para la utilidad de la aplicación. Gracias a nuestro profesor por su enseñanza con Dart nos dió muchas ventajas con el desarrollo, combinando con el framework Flutter, se ve una combinación muy útil para desarrollar cualquier aplicación móvil. Su desarrollo duró unos 2 meses.

Experiencia de aprendizaje


Era la primera vez que utilizábamos Dart pero el profesor nos ayudó mucho, y a parte que la época no fue la mejor (clases online). Muy buen sabor de boca con el lenguaje para desarrollar aplicaciones web junto con Flutter.

Amor Gatuno (En desarrollo)

Descripción


Se pide en la asignatura de Comercio Electrónico desarrollar una página web a alguna empresa/asociación que no la tuviese. Para ello, nuestro grupo decidió contactar con la asociación Amor Gatuno (Valdepeñas).

Una muy buena experiencia, ya que es una práctica en la que se trabaja en grupos de cinco alumnos y eso nos ayuda a sincronizarnos entre varias personas, ya que hasta ahora no había tantos miembros por grupos en prácticas. Es un CMS muy útil para el desarrollo web que ofrece muchas prestaciones.

Sockets

Descripción


Se nos pidió implementar dos versiones del protocolo cliente/servidor TFTP, una sobre el protocolo de transporte UDP y la otra sobre TCP. El objetivo es la comunicación de procesos usando sockets UDP y TCP.

Desarrollo


La práctica se desarrolló en el lenguaje Python (v>=3.3) sobre el sistema operativo UNIX/Linux. Para ello se debería realizar un cliente y un servidor de cada protocolo (archivos .py), con ellos, en los terminales, se ejecutarían y se podrían transmitir mensajes (archivos) con total seguridad.

Experiencia de aprendizaje


Al principio era difícil, debido a que era la primera vez con el lenguaje Python, pero una vez aprendido el lenguaje y el uso de los sockets, se convirtió en un gran desafio. La práctica se tardó bastante tiempo en hacerla, pero al final se obtuvieron unos muy buenos resultados, como se esperaba.
